## Current Project

### Cost Detective

Cost Detective is a Kubernetes analysis dashboard that helps visualize cluster behavior and identify inefficient workloads.

It focuses on questions like:

- Which workloads actually use the resources they request?
- Where are resources wasted?
- How does autoscaling behave under load?
- Where can infrastructure costs be reduced?
