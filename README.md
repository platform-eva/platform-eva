## Current Project

### Cost Detective

Cost Detective is a Kubernetes analysis dashboard designed to visualize cluster behavior and identify inefficient workloads.

The project focuses on answering questions like:

- Which workloads actually use the resources they request?
- Where are resources wasted?
- How does autoscaling behave under load?
- Where can infrastructure costs be reduced?
